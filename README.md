# Currency-changer
A currency converter web service converting 3-currencies:
- NIGERIAN NAIRA (NGN)
- GHANANIAN CEDIS (GHS)
- KENYAN SHILLINGS (KSH)


### Requirements:
* Golang


To run the code locally:

 ```
 $ git clone https://github.com/edwinnduti/currency-changer.git
 $ cd currency-changer
 $ go mod download
 $ go build .
 $ ./main
 ```


To run the Tests:
```
 $ go test -v
```
